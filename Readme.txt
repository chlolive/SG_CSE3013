[Keys]

1. 'l' key를 눌러서 '.txt' 형식의 input file을 선택

2. 't' key를 눌러서 timeline 그리기

3. 's' key를 눌러서 timeline의 보조선 그리기

4. 'c' key를 눌러서 입력 받은 job의 후보들을 보기

5. 'r' key를 눌러서 job scheduling 결과를 확인하기

6. '<-' 또는 '->'와 같은 좌/우 방향키를 눌러서 job scheduling result에서 원하는 job을 targeting하여 찾기

7. 'q' key를 눌러서 메모리 해제 및 프로그램 종료
